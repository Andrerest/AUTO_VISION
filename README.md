# Kinetic Forge

Aplicacion web desarrollada con React, Vite y Tailwind CSS para mostrar inventario automotriz y modificaciones.

## Ejecutar localmente

1. Instala dependencias con `npm install`
2. Configura las variables de entorno necesarias en tu archivo local
3. Inicia el entorno de desarrollo con `npm run dev`

## Scripts disponibles

- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run lint`
